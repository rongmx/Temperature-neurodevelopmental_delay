# Temperature-neurodevelopmental_delay
This repository contains the R code used for the main analyses presented in the article:

**"Early-life heat exposure increases the risk of neurodevelopmental delay in preschool children"**

---

## Overview
This repository provides the code required to reproduce the main statistical analyses of the study.  
The analytical workflow is organized into **four R script files**, corresponding to data preprocessing, model fitting, sensitivity analyses, and figure generation.

All analyses were conducted using **R statistical software (version 4.2.1; R Core Team)**.

Due to data privacy and ethical restrictions, the original study data are **not publicly available**. However, the analytical code and required data structure are fully provided to ensure transparency and reproducibility.
